ᚱ How This System Works
1
Define the Job Role & Required Skills
Enter the job title and list all skills the role requires. These become the benchmark features extracted from resumes using NLP (spaCy/NLTK keyword matching).
2
Add Candidate Resumes
Paste resume text for each candidate. The system parses and cleans the text, extracting mentioned skills and technologies via token matching and lemmatization.
3
ML Scoring Engine Runs
Using TF-IDF vectorization + cosine similarity (scikit-learn), each resume is compared against the job description. A weighted score (0–100) is computed considering skill match %, keyword density, and experience signals.
4
Ranked Results + Gap Analysis
Candidates are sorted by score. Matched skills are highlighted in green; missing required skills are flagged in red — giving recruiters an instant gap analysis.
